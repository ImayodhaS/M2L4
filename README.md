#  Discord Text Translate Bot

Bot Discord berbasis Python menggunakan discord.py yang dapat menganalisis dan menerjemahkan pesan dengan tombol interaktif.

---

## Fitur

-  Analisis teks
-  Terjemahan otomatis
-  Tombol interaktif (Persistent View)
-  Menyimpan riwayat pengguna
-  Balasan private (ephemeral message)

---

## 📌 Cara Menggunakan

Ketik perintah berikut di Discord:
- !start Halo dunia
Bot akan membalas:

Pesanmu diterima! Apa yang kamu ingin aku lakukan?

Dengan dua tombol:
- **Terima Balasa**
- **Terjemahkan Pesan**
